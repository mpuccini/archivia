# Guida all'utilizzo di Archivia

Questa guida ti mostra le principali operazioni che puoi svolgere nel portale.

## 1. Caricare una nuova immagine compilando i metadati

- Vai su "Dashboard" e clicca su "Carica nuovo documento".
- Compila tutti i campi richiesti nei vari step del wizard (ID, titolo, descrizione, tipo, ecc.).
- Carica l'immagine tramite il drag & drop o selezionala dal tuo computer.
- Conferma per salvare documento e immagine.

## 2. Caricare una nuova immagine importando i metadati da file

- Nella dashboard, scegli l'opzione "Importa metadati da file".
- Carica il file (es. JSON, XML, CSV) con i metadati.
- Dopo l'import, associa l'immagine al documento importato.
- Salva per completare l'operazione.

## 3. Caricare una lista di metadati da CSV e upload immagini multiple

- Scegli "Importa lista metadati (CSV)" dalla dashboard.
- Carica il file CSV con i metadati (ogni riga = un documento, Logical ID univoco).
- Dopo l'import, carica le immagini corrispondenti (il Logical ID deve coincidere tra metadato e file).
- Il sistema associa ogni immagine al documento giusto.

## 4. Scaricare metadati e dati

- Puoi esportare i metadati in vari formati (CSV, JSON, XML) dalla dashboard.
- Puoi anche scaricare un documento completo (immagine + metadati) come archivio ZIP.
